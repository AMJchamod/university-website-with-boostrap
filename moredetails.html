Diploma in Cloud Computing & Security

Additional Information

Duration: 720 Hours

Lectures: 4

Quizzes: 80

Students: 120

Available Seats: 120

Price (LKR): 90,000

Module 1: Introduction to Networking

Understanding Networking Fundamentals
OSI Model and TCP/IP Protocol Suite
Network Devices and Infrastructure
Network Security Basics

Module 2: Network Administration

Configuring and Managing Network Devices
IP Addressing and Subnetting
Routing and Switching Concepts
Wireless Networking

Module 3: Introduction to Cloud Computing

Overview of Cloud Computing Models (IaaS, PaaS, SaaS)
Cloud Service Models (Public, Private, Hybrid)
Cloud Deployment Models (Community, Multi-Cloud)
Benefits and Challenges of Cloud Computing

Module 4: Cloud Infrastructure Services

Virtualization Technologies
Cloud Storage Solutions
Compute Resources in the Cloud
Network Configurations in Cloud Environments

Module 5: Cloud Security and Compliance

Security Challenges in Cloud Computing
Identity and Access Management in the Cloud
Data Encryption and Compliance
Security Best Practices for Cloud Environments

Module 6: Networking in Cloud Environments

Virtual Private Cloud (VPC) Configuration
Cloud Networking Services
Interconnecting On-Premises Networks with Cloud Networks
Load Balancing and Content Delivery in the Cloud

Module 7: Cloud Migration Strategies

Assessing Applications for Cloud Migration
Planning and Executing Cloud Migrations
Monitoring and Optimizing Cloud Deployments
Disaster Recovery and Business Continuity in the Cloud

Module 8: Capstone Project

Applying Knowledge to Real-World Scenarios
Designing and Implementing a Networking Solution with Cloud Integration
Presentation and Evaluation of the Project

Practical Labs and Workshops:

Throughout the program, students will have access to hands-on labs and workshops, allowing them to apply theoretical concepts in a simulated environment. Practical exercises will cover networking configurations, cloud service deployments, and troubleshooting scenarios.

Assessment:

Assessment will be a combination of assignments, exams, and the successful completion of the capstone project.

Upon completion of the Diploma in Networking with Cloud Computing, graduates will possess the skills necessary to pursue careers in network administration, cloud architecture, and related IT fields, making them valuable assets in today's technology-driven workplace.
